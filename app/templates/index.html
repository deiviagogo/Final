¿Qué es un Dockerfile?

Un Dockerfile es un archivo de texto que contiene instrucciones para construir una imagen de Docker. Cada instrucción de un Dockerfile crea una capa en la imagen.

Ejemplo básico de un Dockerfile

FROM ubuntu:20.04
RUN apt-get update && apt-get install -y nginx
CMD ["nginx", "-g", "daemon off;"]

Este archivo indica a Docker que use una imagen base de Ubuntu, instale Nginx y lo mantenga en ejecución como proceso principal.
